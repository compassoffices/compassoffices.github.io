Dual-Monitor Setup

Dual-Monitor Workflow

We now use a dual-monitor setup. The live preview stays permanently open on the second monitor — no more auto-refresh delays or manual tapping needed.

Step-by-Step Setup (Do this once)

1

Open & Bookmark the Dashboard

Go to: https://compassoffices.github.io/tools/digital-panel.html
Bookmark this page on the reception computer.

2

Select Your Centre

Choose City → Building → Level

3

Open Live Preview

Click the Preview button next to your level → a new tab with the live display opens.

4

Move to Second Monitor

Drag the preview tab/window to the second monitor.
Hover at the top-right corner → click Toggle Full Screen (green button that appears).

5

Leave It Open Forever

Never close this window/tab. It will update automatically whenever the Google Sheet is edited.

How to Update Company List

Just edit the Google Sheet

In the dashboard → click the level name (e.g. LEVEL 18) → Google Sheet opens → make changes → save automatically → the second monitor updates in a few seconds.

Formatting in Google Sheet (for sub-text)

How to add a smaller line under the company name

Type everything in one single cell exactly like this:

Company Name<td><nobr>smaller text that appears on the next line

Real example:

Advanced Business Alliance Limited<td><nobr>Incorporated in British Virgin Islands and its members' liability is limited

No spaces before or after <td><nobr> — type it exactly as shown.
The smaller text will automatically appear on the next line on the digital panel.

Useful Controls on Preview Page

Hover top-right corner of the second monitor

Two buttons appear:

  • Toggle Full Screen — enter/exit full screen
  • Clear Cache — force instant refresh (use only if update is slow)


Screenshot of the two hover buttons (Toggle Full Screen + Clear Cache)

If Second Monitor Setup is Lost (after restart, etc.)

Quick fix – takes only 30 seconds:
  1. Right-click desktop → Display settings
  2. Drag monitor “2” so it sits exactly at the top-right corner of monitor “1” (position matters)
  3. Click on monitor “2” → set resolution to 1366 × 768
  4. Click Apply → Keep changes


Diagram of correct monitor layout (monitor 2 at top-right of monitor 1)
